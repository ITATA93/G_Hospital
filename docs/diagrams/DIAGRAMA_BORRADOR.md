Este archivo `.drawio` se encuentra **vacio** (sin contenido grafico). Contiene
unicamente la estructura XML base de un archivo draw.io con una pagina en blanco.

Funciona como **espacio de trabajo reservado** para la creacion de futuros
diagramas de procesos hospitalarios.
